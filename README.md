- [Demo](https://fengyuanchen.github.io/vue-number-input)

## Getting started

### Installation

```shell
npm install @chenfengyuan/vue-number-input
```

### Usage

```js
import Vue from 'vue';
import VueNumberInput from '@chenfengyuan/vue-number-input';

Vue.component('number-input', VueNumberInput);
```
